Contains the basic structure for a DDD / CQRS project (optionally event sourced):
* Abstracts and Interfaces for:
    * Aggregates
    * Entities
    * ValueObjects
    * Commands
    * CommandHandlers
    * Events
    * EventSourced Aggregates / EventStores
    * Projections
    * Revisions
    * ...
* Infrastructure:
    * CommandBus
